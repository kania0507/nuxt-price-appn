# Kalkulator VAT - Nuxt 3
Prosty kalkulator VAT stworzony w Nuxt 3 z wykorzystaniem Tailwind CSS, SQLite i Drizzle ORM.  
Umożliwia dodawanie produktów, obliczanie kwoty brutto i podatku VAT oraz przeglądanie zapisanych produktów.

# Technologie
- [Nuxt 3](https://nuxt.com)
- [Vue 3](https://vuejs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [SQLite](https://www.sqlite.org/)
- [Drizzle ORM](https://orm.drizzle.team/)

# Szczegóły
- Walidacja frontend + backend
- Oddzielona logika obliczeń
- API GET/POST
- Możena modyfikować schemat bazy w server/drizzle/schema.ts
- Toasty (powiadomienia)

# TO RUN DEV:
> npm install
> npm run dev
